1.2. 使用方法

由于GitHub并不支持渲染Markdown中Latex数学公式，因此推荐将本项目clone下来后使用编辑器打开

git clone https://github.com/iamNCJ/Discrete_Mathematics_Explained_in_Detail.git

或者直接在我的主页上阅读

https://iamncj.github.io/tags/Discrete-Math/

本项目的几大用途：

  1. 用于自学
  2. 用于预习
  3. 课后复习
  4. 考前复习
  5. 用于科班人士重温CS基础知识
  6. ……

（自认为本项目详细程度完全足够任何人自学）

禁止将本项目用于商业用途！！！
